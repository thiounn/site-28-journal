# site-28-journal

## 项目简介

`site-28-journal` 是一个用于归档与发布独立 HTML 页面的仓库。这里不针对任何特定域名或网站，而是作为一组静态页面的长期存档与展示空间。每个页面均为独立的 HTML 文件，可经由 GitHub Pages 直接访问。

## 目录说明

```
site-28-journal/
├── index.html          # 仓库主页 / 页面索引
├── pages/              # 存放独立 HTML 页面的目录
│   ├── page-001.html
│   ├── page-002.html
│   └── ...
└── assets/             # 共享资源（如 CSS、图片、字体等）
    ├── css/
    ├── images/
    └── ...
```

- `index.html`：页面列表或说明，便于浏览。
- `pages/`：所有独立 HTML 文件均放置于此，文件名按顺序或日期命名。
- `assets/`：页面可能引用的公共资源，按类型组织。

## 页面归档说明

- 每个 HTML 页面均为自包含或依赖 `assets/` 下资源的独立文档。
- 页面内容可能涉及不同主题、笔记或实验性项目，彼此之间无强制关联。
- 归档目的仅为保存与分享，不承诺更新频率或内容方向。
- 所有页面均以静态方式部署，无需后端服务。
